# Flere eksempler på hva stilguider kan brukes til:

På denne branchen finner du noen flere eksempler på hvordan en stilguide kan brukes.

For å kjøre stilguiden kjør kommandoen: 

```html
npm run styleguide-server
```

Deretter gå til:
```html
http://localhost:3000
```
